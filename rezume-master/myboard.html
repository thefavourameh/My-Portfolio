MyBoard

About App

MyBoard is a web-based task management application designed to simplify personal to-do's and tracking for individuals. It provides a user-friendly platform for organising tasks, setting deadlines, tracking progress, and prioritising activities.

Features

  • User Authentication and Security: User registration, login, Password encryption and Session management
  • Dashboard View: Displaying tasks, deadlines, progress and priority levels, quick access to assigned tasks, upcoming deadlines and completed tasks.
  • Task Management: List creation, quick and detailed task creation, update task, task tracking, and delete task.

Technologies Used

  • Frontend: Reactjs, TailwindCSS, JavaScript
  • Backend: Java, Spring Boot
  • Database: PostgreSQL
  • Authentication: JSON Web Tokens (JWT)
  • Deployment: Docker, CI/CD
